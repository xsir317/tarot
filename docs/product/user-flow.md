# 用户流程图

## 完整用户旅程（简化版）

```
首页 → 问题输入页 → 问题验证
                      ↓ 问题适合
                洗牌动画 → 抽牌动画 → 翻牌互动
                                              ↓ 翻完3张
                                         检查额度
                                              ↓ 额度充足
                                         解读结果页
                                              ↓
                                         再次占卜
```

## 关键页面

### 首页
- 品牌 Logo
- "开启你的心灵指引之旅"
- [开始占卜] 按钮

### 问题输入页
- 问题输入框（多行文本）
- 性别选择（可选）
- [开始抽牌] 按钮

### 翻牌互动页
- 3 张牌背，依次可点击翻转
- 翻牌后逐张展示解读
- [查看整体解读] 按钮

### 解读结果页
- 3 张牌的解读
- 整体解读
- [再次占卜] 按钮
- [分享] 按钮
- [用户中心] 按钮

### 注册引导弹窗
- "您的免费次数已用完，注册即可继续使用"
- [手机号注册] 按钮
- [邮箱注册] 按钮
- [继续按次付费] 按钮

### 付费页面
- [按次付费 $0.99-1.99]
- [包月订阅 $9.99/月]
- [自愿打赏]

### 用户中心
- 用户信息
- 额度信息
- 订阅状态
- 历史记录
- [退出登录]
